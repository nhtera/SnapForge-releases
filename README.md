# SnapForge Releases

Auto-update feed and release assets for [SnapForge](https://github.com/nhtera/SnapForge) — a native macOS screenshot, recording, and annotation app.

## Download

Download the latest version from [Releases](https://github.com/nhtera/SnapForge-releases/releases).

## Auto-Update

SnapForge uses [Sparkle](https://sparkle-project.org/) for auto-updates. The appcast feed is hosted in this repository.

**Feed URL:** `https://raw.githubusercontent.com/nhtera/SnapForge-releases/main/appcast.xml`
